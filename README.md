# Netherrack

A discord bot for managing vanilla Minecraft servers

## Features

- [X] Discord <-> Minecraft chat bridge
    - [X] Public chat bridge
        - [X] Minecraft -> Discord
            - [X] Webhooks
        - [X] Discord -> Minecraft
        - [X] Death messages bridge
    - [ ] Private message bridge
    - [ ] Online players in channel description
- [ ] Discord <-> Console bridge
    - [X] Mirror console to discord
    - [X] Chat in bridge channel to send console commands
- [ ] Online players in status
- Commands
    - [ ] Server info 
        - Players Online
        - Uptime
        - MOTD and Icon
    - Management
        - [ ] `/run` console commands
        - [ ] `/start`, `/restart` and `/stop`
